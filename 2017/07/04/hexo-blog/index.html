搭建静态blog

1、 安装全局hexo-cli

  • npm install hexo-cli -g

2、生成blog项目

  • hexo init blog(项目名)

3、进入 blog项目文件夹中

  • cd blog

4、进行安装依赖

  • npm install

5、启动服务

  • 本地查看， 在当前 项目 根目录打开 cmd ，输入以下命令，按提示操作进行,如有改动 刷新页面即可，修改配置文件时，须重启服务。
    • hexo server

6、 github 部署

  • github建库： 名字.github.io

    • username.github.io

  • 发布github 需要一个发布github的插件

    • npm install hexo-deployer-git –save

  • 找到blog–>_config.yml–>

    • deploy:
      • type: git
      • repo: https://username:password@github.com/username/username.github.io.git
      • branch: master
  • 主页面–>settings–>email–> 去掉这项 Keep my email address private

7、生成上传及发布

每次更新都要执行下面两部

  • 当前 项目 根目录下打开 cmd ，输入以下两条命令，按提示操作进行
  • hexo g 生成
  • hexo d 发布

8、主题

  • git clone 主题文件地址
  • blog–>_config.yml–>theme: themes下的主题文件名换成最新的主题文件名
