  • 不能用汉字
  • 首写不能为数字

    • login_info login-info √

相对路径和绝对路径

是指在某个文件中引入文件需要写路径

绝对路径：根据一段路径直接能到想要找的目标

例如：ip域名D：\2017年\3月

相对路径：从一个参照物出发，去找另一个目标文件

  • 寻找上一级的时候写../
  • 平级文件之间直接写名称
  • 向下查找的时候写“文件夹/子文件夹/子文件.后缀名”

    在工作中一般使用相对路径

    表单

    <form> 标签用于创建供用户输入的 HTML 表单。
    <form> 元素包含一个或多个如下的表单元素

    |表单元素|描述|
    |:–:|:–:|
    |input|规定了用户可以在其中输入数据的输入字段|
    |textarea|文本区域|
    |button|按钮|
    |select|用来创建下拉列表|
    |option|选项|
    |optgroup|选项组|
    |fieldset|可以将表单内的相关元素分组|
    |label|为 input 元素定义标注（标记）|
