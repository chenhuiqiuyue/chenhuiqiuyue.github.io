echarts

其他

  • init 初始化
    • 参数 dom 实例容器
    • theme 应用的主题。
    • opts
      • devicePixelRatio
        设备像素比，默认取浏览器的值window.devicePixelRatio
      • renderer
        渲染器，仅支持’canvas’。
      • width
        可显式指定实例宽度，单位为像素px。
      • height
        可显式指定实例高度，单位为像素px。
  • connect 多个图表实例实现联动。
  • disconnect 解除图表实例的联动
  • dispose 销毁实例，实例销毁后无法再被使用。
  • getInstanceByDom 获取 dom 容器上的实例
  • registerMap 注册可用的地图 【包括 geo 组件或者 map 图表】
  • registerTheme 注册主题，
  • graphic 图形相关帮助方法。

异步加载 通过 jQuery 等工具异步获取数据后通过 setOption 填入数据和配置项就行。

echartsInstance 通过 echarts.init 创建的实例。

action ECharts 中支持的图表行为，通过 dispatchAction 触发。

  • highlight Action 高亮指定的数据图形。
  • downplay Action 取消高亮指定的数据图形。
  • legend 图例组件相关的行为，必须引入图例组件后才能使用
  • tooltip 提示框组件相关的行为，必须引入提示框组件后才能使用。
  • dataZoom 数据区域缩放组件相关的行为，必须引入数据区域缩放组件后才能使用。
  • visualMap 视觉映射组件相关的行为，必须引入视觉映射组件后才能使用。
  • timeline 时间轴组件相关的行为，必须引入时间轴组件后才能使用。
  • toolbox 工具栏组件相关的行为，必须引入工具栏组件后才能使用。
  • pie 饼图相关的行为，必须引入饼图后才能使用
  • geo 地图组件相关的行为，必须引入地图组件后才能使用。
  • map 地图图表相关的行为，必须引入地图图表后才能使用。
  • graph 关系图 相关的行为，必须引入 关系图 后才能使用。
  • brush 区域选择相关的行为。

    events 在 ECharts 中主要通过 on 方法添加事件处理函数，该文档描述了所有 ECharts 的事件列表。 也有鼠标事件

  • 鼠标事件 ‘click’，’dblclick’，’mousedown’，’mouseup’，’mouseover’，’mouseout’，’globalout’，’contextmenu’。
  • legendselectchanged Event ACTION: legendToggleSelect 切换图例选中状态后的事件
  • legendselected Event ACTION: legendSelect 图例选中后的事件
  • legendunselected Event ACTION: legendUnSelect 图例取消选中后的事件。
  • legendscroll Event ACTION: legendscroll 图例滚动事件。
  • events.datazoom Event ACTION: dataZoom 数据区域缩放后的事件。
  • datarangeselected Event ACTION: selectDataRange 视觉映射组件中，range 值改变后触发的事件。
  • timelinechanged Event ACTION: timelineChange 时间轴中的时间点改变后的事件。
  • timelineplaychanged Event ACTION: timelinePlayChange 时间轴中播放状态的切换事件。
  • restore Event ACTION: restore 重置 option 事件。
  • dataviewchanged Event 工具栏中数据视图的修改事件。
  • magictypechanged Event 工具栏中动态类型切换的切换事件。
  • geoselectchanged Event ACTION: geoToggleSelect geo 中地图区域切换选中状态的事件。用户点击选中会触发该事件。
  • geoselected Event ACTION: geoSelect geo 中地图区域选中后的事件。
    -vents.geounselected Event ACTION: geoUnSelect geo 中地图区域取消选中后的事件。
  • pieselectchanged Event ACTION: pieToggleSelect series-pie 中饼图扇形切换选中状态的事件
  • …等事件

    events 理解

  • ECharts，一个纯 Javascript 的图表库，可以流畅的运行在 PC 和移动设备上，兼容浏览器【IE8/9/10/11，Chrome，Firefox，Safari等】，底层依赖轻量级的 Canvas 类库 ZRender，提供直观，生动，可交互，可高度个性化定制的数据可视化图表。ECharts 3 中更是加入了更多丰富的交互功能以及更多的可视化效果，并且对移动端做了深度的优化。

  • 图表类型有:折线图，柱状图，散点图，饼图，K线图，用于统计的盒形图 ; 地理数据可视化的地图，热力图，线图;关系数据可视化的关系图，treemap，多维数据可视化的平行坐标，还有用于 BI 的漏斗图，仪表盘，并且支持图与图之间的混搭

  • 动态数据

  • 绚丽的特效

  • ECharts，一个纯 Javascript 的图表库，可以流畅的运行在 PC 和移动设备上，兼容当前绝大部分浏览器（IE8/9/10/11，Chrome，Firefox，Safari等），底层依赖轻量级的 Canvas 类库 ZRender，提供直观，生动，可交互，可高度个性化定制的数据可视化图表。

  • ECharts 3 中更是加入了更多丰富的交互功能以及更多的可视化效果，并且对移动端做了深度的优化。

  • 丰富的图表类型
    ECharts 提供了常规的折线图，柱状图，散点图，饼图，K线图，用于统计的盒形图，用于地理数据可视化的地图，热力图，线图，用于关系数据可视化的关系图，treemap，多维数据可视化的平行坐标，还有用于 BI 的漏斗图，仪表盘，并且支持图与图之间的混搭。

  • 你可以在下载界面下载包含所有图表的构建文件，如果只是需要其中一两个图表，又嫌包含所有图表的构建文件太大，也可以在在线构建中选择需要的图表类型后自定义构建。
