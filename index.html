    6、html简介

    叫做超文本标记语言，是使用标签来进行内容的标记，每一个标签都有自己特殊的意义
    文本：普通的文字
    html 这是普通文本
    超文本：由尖括号和特定的文本组成的
    <html>'这是超文本
    超文本不会显示在页面中

html是文本语言：是由符号和文本组成，里面不能含有图片等内容
html文件的运行环境是 — 浏览器

7、什么是标签？

标签是由尖括号和特定的文本组成的
特定文本叫做标签名
例如 :

<p>这是p标签

标签的分类：

开始标签><开始标签>
结束标签 ></结束标签>
自闭合标签【空标签】 ><自闭合标签 />可以省略”/”

8、标签属性

写在开始标签或空标签的标签名后面，用空格隔开，一个标签可以有多个标签属性，例如：

<p style="color:red;">，
<img align="center" width="100" />
标签属性：是由键值对组成的，有属性名和属性值，属性名和属性值之间用等号连接，属性值用引号包裹，多个标签属性之间用空格隔开

9、元素

由标签和内容组成的，例如：

<p>我是一个段落标签，里面放着一段文字</p> p元素

元素可以内容为空

<p></p> p元素
空标签也是元素
<img /> img元素

元素的嵌套：

有开始标签和结束标签组成的元素是可嵌套的元素

1.<div> 2.<p> 3.<img /> 4. </p> 5. <img /> 6.</div>
空标签不能进行嵌套
1.<img />

9、html中常用的元素以及元素的分类

元素的分类

  • 块级元素：独占一行
    |标签名|描述|
    |:–:|:–:|
    |div|是一个无意义的标签，一般用来进行结构的划分|
    |ul|无序列表|
    |ol|有序列表|
    |li|列表项|
    |dl|定义列表|
    |dt|定义名称|
    |dd|定义描述|
    |p|段落|
    |hr|分割线|
    |form|表单 input|
    |h1-h6|标题（文章 专题 栏目）|
    |table|表格 tr th td|

格式重点：
(1)、ul和ol元素中第一层级嵌套的元素永远是li标签
(2)、dl中第一层级嵌套的元素只能是dt和dd
(3)、p标签中不能嵌套自己或者其他块级元素

  • 行内元素（内联元素）：在一行显示
    |标签名| 描述|
    |:–:|:–:|
    |span|无意义标签，进行小结构的描述|
    |b|文本加粗|
    |strong|文本加粗 语气强调（是浏览器识别的)|
    |i|文本斜体|
    |em|文本斜体 语气强调（是浏览器识别的）|
    |br|强制换行|
    |sub|下标|
    |sup|上标|
    |font|已经弃用 字体|
    |s|弃用 中划线|
    |strike|中划线|
    |del|删除线|
    |u|下划线|
    |img|引入图片|
    |input|表单框|

    行内元素基本上都是进行文字控制的元素，不会进行结构的划分

    标签属性之alt和title的区别

  • alt 应用正在img标签中，如果图片加载出现问题，alt中的文字就会显示在该图片位置上
  • title任何元素都可以使用，鼠标放置在这个元素上的时候，会出现一个跟随鼠标的文本框，title中的文字就会在这个文本框中

10.html的基本结构

1、<!- -html的注释 用户看不到 编辑人员使用- ->

2、<!DOCTYPE html> 是html5的文档声明

  • <!–文档声明
    • 文档声明不是html标签
    • 文档声明必须写在html所有标签和内容之前
    • 作用：浏览器从上到下读取代码，根据文档声明来判断应该使用html版本几的规则来读取这个html文件
    • html4的文档声明
      <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
  • 高版本永远会向下兼容
    • html5的文档声明下的文件可以继续使用html4的标签和标签属性等
    • html4的文档声明下的文件不支持html5的所有标签和标签属性

      3、 <html></html>

  • html元素是整个文件的根元素 root element
  • html元素是块级元素
  • html元素告知浏览器整个页面开始

    4、<head></head>

  • head元素是html文件的头部
  • head标签中的内容不会出现在浏览器的可视窗口中

    5、<meta charset="UTF-8">

  • meta 文件的元信息
  • charset 文件编码
  • UTF-8 国际通用编码
  • GBK 中国编码
  • gb2312
  • keywords ： 关键词 每一个关键词（字）用逗号隔开
  • description ： 网页描述
  • 这些内容都是给浏览和搜索引擎看的

7、引入一个js外部文件

8、<body></body>

  • 这个标签中放置的是html所有展示给用户的内容
  • 这个标签这中可以嵌套所有的标签

搭建静态blog

1、 安装全局hexo-cli

  • npm install hexo-cli -g

2、生成blog项目

  • hexo init blog(项目名)

3、进入 blog项目文件夹中

  • cd blog

4、进行安装依赖

  • npm install

5、启动服务

  • 本地查看， 在当前 项目 根目录打开 cmd ，输入以下命令，按提示操作进行,如有改动 刷新页面即可，修改配置文件时，须重启服务。
    • hexo server

6、 github 部署

  • github建库： 名字.github.io

    • username.github.io

  • 发布github 需要一个发布github的插件

    • npm install hexo-deployer-git –save

  • 找到blog–>_config.yml–>

    • deploy:
      • type: git
      • repo: https://username:password@github.com/username/username.github.io.git
      • branch: master
  • 主页面–>settings–>email–> 去掉这项 Keep my email address private

7、生成上传及发布

每次更新都要执行下面两部

  • 当前 项目 根目录下打开 cmd ，输入以下两条命令，按提示操作进行
  • hexo g 生成
  • hexo d 发布

8、主题

  • git clone 主题文件地址
  • blog–>_config.yml–>theme: themes下的主题文件名换成最新的主题文件名
